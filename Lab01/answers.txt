Question 1:
If the user chooses "Cancel" (or closes the dialog box), the variable option will receive a value that does not equal JOptionPane.YES_OPTION. Based on the ternary operator in the sample code, the program will display a message saying "You've chosen: No".
Question 2:
To show only "Yes" and "No": We can change the optionType parameter in showConfirmDialog to JOptionPane.YES_NO_OPTION.
To customize button labels (e.g., "I do" / "I don't"): We should use the showOptionDialog method. This method allows us to pass an array of Object (Strings) to define custom labels for the buttons.

Section 6.4:
A year is a leap year if it is divisible by 4. However, years divisible by 100 are not leap years unless they are also divisible by 400. For example, 1800 is a common year, but 2000 is a leap year.
